引言：占卜的核心——“动”

经过交流，很多学者问我，为什么奇门遁甲有时候起局很容易，但就是断不好，甚至断不准？第一句对了，后面就全错了。

这其中的关键，就在于奇门讲究一个“动”字。

奇门是一门“时空学”，你必须要有“动”，才能进入它所模拟的时空之内。这个“动”，不是自身的动，用周易的术语来说，就叫“外应”。

你必须抓住外部世界的一个“动”（某个动作、动向、动机），才能真正进入九宫八卦这个时空模型。什么是时空？时就是时间，空就是空间（方位），在奇门里最明显的就是九宫八卦。

如果没有“动”，你就进不了这个时空。

【整理后版本 3】:

很多人用网络软件起局，有时候准，有时候不准，就是因为你发现不了这个“外应”，这个“动”。

这个“动”到底是什么呢？

举个例子，我跟朋友在一起，他忽然在路上摔了一跤。这个“摔跤”，就是一个“动”，一个“外应”。我就可以以这个“动”为切入点，进入奇门时空，提取信息。

如果你没有抓住这个“动”，只是按部就班地用月干代表朋友，或者用他的年命入局，那么你抓出来的信息就太少了，抓不住核心。偶尔巧了能断准，但反复应用，准确率会非常低。

没有“动”这个外应，你就进不了真正的时空。

【整理后版本 4】:

河南有个朋友，为同一件事，前后起了两次局，结果完全相反。为什么？就是因为没有抓住“动”。

记住，我们用任何术数（周易、奇门、六壬、太乙、梅花等）进行占卜，都必须讲究一个“动”。

动则发用。

必须有“动”，你才能提取出太极点，才能抓住那个特定的时空。没动，不敢占。

【整理后版本 5】:

并不是说，你今天忽然想起来一个多年未见的朋友，心里想看看他怎么样，就起个局来测。这样测是不准的。

你心里的“想”，是你自己的“动”，这是一个前提。但还必须要有外部的“动”（外应），你才能真正进入并抓住那个时空运行的轨道。

奇门遁甲的时空，就是九宫八卦所代表的时间（节气）和空间（方位）。你没有外应这个“钥匙”，就进入不了这个时空学模型，占断自然就不灵验。这是正常现象。

【整理后版本 6】:

占卜原则一：有疑则占，无疑不占

占卜还必须讲究一点，就是你心里必须有一个疑惑点，这个疑惑点才能产生你自己的“动”。

古人说：“有疑则占，无疑不占。”

意思就是，你心里真的有疑问了，才去占卜；没有疑问，就不要去算卦。

比如有个人，他心里没什么事，就是出于好奇，或者想试探一下，随便问问财运怎么样。这种情况就属于“无疑而占”，占出来的结果往往不准，甚至与实际情况相反，你千万别信。

只有你心里真的有疑惑，比如家里出了忧心的事，不知道怎么办了，这时候你去占，才应该按照卦象的指示去做。

【整理后版本 7】:

占卜原则二：首卦为准，事不过三

另外一个规矩是，占卜以第一次为准。

为同一件事情占卜，不要超过三次，所谓“事不过三”。你反复地去占，只有第一次的结果是最准的。

【整理后版本 8】:

1. 必须有“动”，也就是“外应”。
2. 必须心有疑惑，即“有疑则占”。
3. 以第一卦为准。
4. 心要虔诚。

学习奇门的心态与德行

古时候占卜是非常神秘和严肃的，占卦前都要做好几天的准备，要虔诚、要上香，不是开玩笑。

所以，我们不要把奇门遁甲这门学问，当成一个开玩笑的工具。有些人跟我说“玩奇门”，我听着很不高兴。这是古人传承下来的学问，一定有它的用处，我们要潜心去研究、去挖掘，而不要轻视它。

奇门的流派：转盘与飞盘

在农村，奇门遁甲常被称为“奇门遁”。

我听爷爷他们说，奇门主要流传在山东和河南地区。据我了解，盘式永远是一样的，要么转盘，要么飞盘。

【整理后版本 16】:

  • 转盘奇门：主要流传在山东、河北这一带。山东有传承的奇门，都是转盘。
  • 飞盘奇门：主要流传在河南一带。

奇门的传承主要在黄河南北地带，像西藏、新疆那边根本就没有。

这就像武术的南北差异，北方注重螳螂拳、六合拳，南方是南拳、咏春拳。转盘和飞盘各有特点，只要应用好了都一样。

【整理后版本 17】:

转盘奇门的原理

奇门遁甲的盘式，是由地、天、人、神四盘构成的。

  • 地盘：为地理，是基础。
  • 天盘：模拟天体的运转。天上的行星都在转动，所以天盘也要转动。
  • 人盘：遵循“天人合一”的概念。既然天盘是转动的，那么人盘也要跟着转动。
  • 神盘：是隐藏在天地之间的一种物质，既然天地都转，那么神也必须跟着转。

这就是转盘奇门遁甲的基本原理。

【整理后版本 18】:

转盘与飞盘的区别

  • 转盘：模拟的是天体行星的转动。
  • 飞盘：模拟的是宇宙间磁场的流动。

为同一件事占卜，转盘和飞盘断出来的结果应该是一样的，只是看待事物的角度不同。
* 转盘是围着圈去看。
* 飞盘是上下去看。

至于哪家更正宗，谁也不敢说。

【整理后版本 19】:

奇门的应用：占卜与运筹

1. 占卜的核心：三传

用奇门占卜一个事物，主要就是看这个事物的发展状态。这就离不开我们前面讲的“三传”。

初传（天乙宫） -> 中传（值符宫） -> 末传（值使宫）

一般人来求测时，他所处的状态，往往就在这三传的某个阶段：
* 可能在初传状态（事情刚刚开始）。
* 可能在中传状态（事情正在演变）。
* 可能在末传状态（事情马上要结束）。

所以，要想用奇门侦测一个事情的来龙去脉，你必须用三传。

【整理后版本 20】:

2. 运筹的核心：择吉与主客

运用奇门遁甲进行运筹，在理数层面，其实就是两件事：
1. 择吉：选择好的时间、好的方位。
2. 定主客：判断什么时候该主动，什么时候该被动。

这就是运筹，为了更好地办事，趋吉避凶。

至于念咒、画符、摆阵、破阵，那些属于法术奇门的范畴。真正的法术奇门是阵法，需要供奉阴坛，才能请神指挥，改变磁场。社会上教的那些大多是假的。

核心断法一：年命落宫分析

我们来看他的年命“丁”在天盘上的落宫——乾六宫。

1. 综合信息解读

  • 星：临天禽星（老师口误为天离星）。天禽星为中宫之星，主正直。
  • 宫：落乾宫，乾为天，为首领，代表有本事。
  • 神：上乘螣蛇，主烦恼、忧虑、压力大。
  • 将：临亥将（登明），主困难、缓慢。
  • 空亡：乾宫临日空亡（壬申日戌亥空），主迷茫。

2. 生克关系分析

  • 年命：“丁”，五行属火。
  • 落宫：“乾六宫”，五行属金。
  • 关系：火克金，即“年命克宫”。

年命克宫的象意：代表此人需要消耗自身能量去克制环境，说明付出比较大。

综合判断（当前运势）：
这个人为人正直、有本事，但目前运势不佳。他正处于一种有烦恼、有困难、前途迷茫的状态，感觉压力很大，付出很多。

【整理后版本 4】:

3. 格局分析

我们再看年命宫的格局：“丁下临癸”。

  • 格局名称：“朱雀投江”，是凶格。
  • 原理：天盘“丁”（阴火）与地盘“癸”（阴水），是同性质相克（水克火），而且是“以下克上”。

格局解读：
“朱雀投江”代表烦恼忧虑、不知所措、仿佛要投降了一样没办法。

最终判断（当前运势）：
综合所有信息，这个人现在的状况十分不好。

年命与落宫五行关系总结

  • 年命生宫：主付出。
  • 年命克宫：主付出更大，因为要消耗自身去克制。
  • 宫生年命：大吉大利，主有外界帮助。
  • 宫与年命比和：也比较好。

我们再看休门的引干。
* 休门原始在坎一宫，该局坎一宫的地盘是“己”。
* “己”是当天的时干，也可能是年干（如果乙未年按己未看），代表工作、长辈或子女。

综合判断（烦恼来源）：
他现在的烦恼和压力，主要来源于家庭内部，并且可能与工作不顺心、为长辈操心、为子女操心等事情有关。

5. 运筹决策分析

占卜个人运势，最简单也最重要的方法，就是用年命天盘干落宫来分析。

那么，如何给他决策建议呢？比如他想去投资，利不利于他？这就需要看主客关系。

  • 天盘为客（代表求测者）。
  • 地盘为主。

分析年命宫（乾六宫）：
* 天盘星是天禽星（土）。
* 地盘是乾宫（金）。
* 土生金，天盘生了地盘。

决策：
天盘（客）生了地盘（主），说明他自己付出了，但没有得到，能量被主方（环境、对手等）吸收了。这叫“利主不利客”。

所以，非常不利于他。他现在运势十分糟糕，格局又是“朱雀投江”，什么都不能干，千万不能去投资！

核心断法二：流年运势分析

如何看他这一年的流年运势好坏？

方法：

看年干（太岁）落宫与年命落宫的生克关系。

生克关系解读：
* 年命宫克太岁宫：最凶，为“犯太岁”，以下犯上，必有小灾、破财。
* 太岁宫克年命宫：也为不顺，但凶性比上一种稍小。
* 年命宫生太岁宫：主今年付出大，开支大，可能会生病。
* 太岁宫生年命宫：吉，主今年流年不错，会得到外界的帮助。
* 年命宫与太岁宫比和：主今年平平，没什么大事。

流年占测的要点：

  • 问则有，不问则无：他问哪一年，你就看那一年的年干落宫。

  • 以我们的实例分析今年的流年：

    • 年干（太岁）是“己”。
    • “己”落在坎一宫。
    • 年命“丁”在乾六宫。
    • 坎一宫（水）与乾六宫（金）是金生水关系，即年命宫生太岁宫。
    • 结论：主今年付出大。

  • 再看太岁宫（坎一宫）的信息：

    • 临天蓬星（主争斗），说明今年谨防有争斗。
    • 宫内有地盘“戊”，有“戊癸合”，说明可能与钱财合作有关。

综合判断：今年流年不顺，付出大，且易因钱财合作之事发生争斗。

核心断法三：特殊状态判断（入墓、击刑、旬空）

1. 年命入墓

如果年命天干落入了“墓”库宫位，代表此人有力使不上，心有余而力不足，是失败的象征，时机不成熟。必须等到流年流月来“冲墓”的时候，运势才可能会好。

三奇六仪入墓宫位：
* 乙 -> 坤二宫
* 丙、戊 -> 乾六宫
* 丁 -> 艮八宫
* 庚、癸 -> 坤二宫
* 壬 -> 巽四宫

(注：此处老师口误，乙的墓库在戌，应为乾六宫)

如果测恋爱，双方年命都入墓了，那这段感情肯定成不了。

1. 年命是“戊”。
2. 戊为阳土，长生在寅。我们顺着排十二长生...
3. 排到“子”地时，正好是“胎”。
4. 我们看奇门局中，天盘“戊”正好落在坎一宫，坎宫地支就是“子”。

结论：年命“戊”落在了“胎”地，说明此人确实怀孕了。

【整理后版本 4】:

核心断法二：判断胎儿性别

确定怀孕后，我们就不用再看年命宫了，转而看坤二宫。

坤宫为母亲，坤宫内的信息就代表了胎儿的信息。

判断男女，主要依据坤宫内的九星、天干和八门的阴阳属性。

阴阳属性划分：
* 九星：天蓬、天任、天冲、天辅为阳星；天英、天芮、天柱、天心为阴星。
* 八门：休、生、伤、杜为阳门；景、死、惊、开为阴门。
* 天干：甲、丙、戊、庚、壬为阳干；乙、丁、己、辛、癸为阴干。

【整理后版本 5】:

判断性别的步骤

  1. 第一步：看星与干
    • 先看坤宫的天盘星是阳星还是阴星。
    • 再看该星所带的天干是阳干还是阴干。
    • 如果星与干的阴阳属性相同（阳+阳 或 阴+阴），则直接以此属性定男女（阳为男，阴为女）。
  2. 第二步：看门
    • 如果星与干的阴阳属性不同（一阳一阴），则星和干就决定不了性别。
    • 这时候，就直接看人盘八门的阴阳属性来决定。

实例分析

看坤二宫：
* 天盘星：天蓬星，是阳星。
* 天盘干：丁，是阴干。
* 星与干的属性不同（一阳一阴），所以它们决定不了。
* 看八门：休门，是阳门。

结论：门说了算，所以这个孩子是男孩。很简单。

【整理后版本 6】:

核心断法三：判断生产是否顺利

判断生产是否顺利，主要看坤宫内地盘和天盘的生克关系。

  • 地盘（坤宫本身）：代表母亲，五行永远是土。
  • 天盘星：代表胎儿（子）。

生克关系解读：
* 地盘克天盘（母克子）：主产速，生产迅速。
* 天盘生地盘（子恋母）：主产迟，生产会慢。
* 天盘克地盘（子克母）：主母亲凶，对母亲不利。
* 地盘克天盘星 (此处老师口误，应为克制太过): 主孩子凶，有危险。

实例分析

  • 地盘（母亲）：坤宫，土。
  • 天盘星（胎儿）：天蓬星，水。
  • 关系：土克水，地盘克天盘。

结论：主产速，生产会比较快。

【整理后版本 7】:

特殊状态判断

  • 如果天盘星所带的奇仪入墓或逢空，都是不吉之象，孩子容易在胎里出问题。

核心断法四：判断产期

方法：

看坤宫的对冲宫位（艮八宫）的天盘干。

实例分析

  1. 坤二宫的对冲宫是艮八宫。
  2. 艮八宫的天盘干是“癸”。

结论：产期可能在癸月或癸日。

（具体是月还是日，需要根据怀孕多久等实际情况来定。）

比如当时是庚辰月，那么辛巳、壬午，到癸未月，也就是两个月以后生产。

【整理后版本 8】:

完整思路复盘与总结

我再把整个思路给大家梳理一遍：

  1. 判断是否怀孕：看求测人的年命或日干，是否落在了十二长生中的“胎”地。
  2. 判断胎儿性别：
    • 看坤宫的天盘星和天干的阴阳属性。
    • 如果相同，则以此属性定男女。
    • 如果不同，则看八门的阴阳属性定男女。
  3. 判断生产顺利否：
    • 看坤宫地盘（母）与天盘星（子）的生克关系。
    • 地盘克天盘 -> 产速
    • 天盘生地盘 -> 产迟
    • 天盘克地盘 -> 母凶
    • 地盘克天盘太过 -> 子凶
    • 天盘奇仪入墓或空亡 -> 不吉
  4. 判断产期：看坤宫的对冲宫的天盘干，应在相应的月或日。

注意：用坤宫判断性别，前提是正在怀孕中。如果孩子已经生下来了，就不能再用这个方法了。

核心技法一：大运的排法

推演终身运势，也就是“大运”，主要依靠地盘的六仪（戊己庚辛壬癸）来划分时间阶段。

排大运的步骤：
1. 找出出生年的旬首：
* 需要知道求测人的出生年干支。
* 用“年地支顺序数 - 年天干顺序数”的方法，找出该年所属的旬首。
2. 确定大运起点：
* 看该旬首所遁藏的天干（戊己庚辛壬癸）在地盘的哪个宫位。
* 该宫位就是1-10岁的大运起点。
3. 依次排布大运：
* 从起点宫开始，按照“戊己庚辛壬癸”的顺序，结合九宫飞布的路径（阳遁顺飞，阴遁逆飞），依次排布后续的大运。
* 每个宫代表10年。
* 特殊规则：中五宫代表20年。

核心技法二：判断一生及当前运势

在分析大运之前，我们先从日柱和时柱看一个总体的趋势。

  • 日柱：代表当前的运势。
  • 时柱：代表一生的运势。

判断方法：看干支的生克关系。干为自己，支为运势。
* 支克干：不好，运势克自己。
* 干克支：自己克运势，付出大。
* 支生干：大吉大利。
* 干生支：自己生运势，付出大。

断局详解一：分析一生及当前状态

以本局为例：

结论：他当前运势不顺，而一生来看，是付出大的命。
