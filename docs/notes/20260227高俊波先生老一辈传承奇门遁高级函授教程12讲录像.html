什么是“克应”？
克应就好比是“格局”。比如：
* 在太乙神数中，“太乙与文昌同宫”就是一个克应，主文将造反。

十二将神与星、门、奇仪同宫时，也会产生特定的克应关系，反映出不同的情况。

掌上小奇门（掐指神算）用法详解

现在我们重点讲一下掌上小奇门的用法。

1. 手掌布局：
在手掌上将十二地支（子、丑、寅...亥）按照九宫的位置固定下来。

  • 食指根 -> 巳
  • 食指中 -> 午
  • 食指尖 -> 未
  • 中指尖 -> 申
  • 无名指尖 -> 酉
  • 小指尖 -> 戌
  • 小指中 -> 亥
  • 小指根 -> 子
  • ...以此类推，将十二地支排布在指节上。

2. 起法：
这是一个“掐指神算”的过程，分为三步：
* 第一步：从年上起月。
* 找到当年地支（如庚寅年，就从“寅”位）作为正月。
* 然后顺数，一直数到求测的月份，看落在哪个地支上。
* 第二步：从月上起日。
* 从上一步落下的地支开始，作为初一。
* 然后顺数，一直数到求测的日期，看落在哪个地支上。这个地支所代表的将神，就代表了求测人的状态。
* 第三步：从日上起时。
* 从上一步落下的地支开始，作为子时。
* 然后顺数，一直数到求测的时辰，看落在哪个地支上。这个地支所代表的将神，就代表了所测之事的状态。

3. 判断：
最后，看代表事的将神与代表人的将神之间的五行生克关系，来判断事情的吉凶成败。

* 从“卯”位起子时，顺数到午时，落在“酉”位。

* 事：酉，五行属金。
* 关系：金克木，“事”克“人”。
